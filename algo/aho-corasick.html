Aho-Corasick Algorithm

Category: Multiple Keyword Search

Description: Builds a Trie out of multiple patterns and uses BFS with failure links to perform simultaneous pattern matching in linear time.

📦 Business Case:

  • Efficiently matches user search terms like "action, betrayal, sci-fi" in movie descriptions.
  • Backend search engines to find content with multiple features or genres simultaneously.
  • Auto-tagging or classifying video metadata based on detected keywords.
🧠 How It Works?
  • Build a Trie with all search patterns.
  • Link nodes using BFS-based failure links to fallback on mismatch.
  • Scan the text in one go and track all matches simultaneously.

🧮 Complexity Analysis
  • Time: O(N + L + Z)
  • Space: O(L * Σ), where L = total length of patterns, Σ = alphabet size
