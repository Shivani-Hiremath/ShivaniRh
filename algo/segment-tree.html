Segment Tree

Category: Data Structures, Range Query

Description: A tree-based data structure that allows efficient range queries and updates in logarithmic time.

📦 Use Cases (AWS Context):

  • Real-time log analytics using AWS Lambda or Kinesis Firehose.
  • Efficient range-sum or max query over IoT data streams in AWS IoT Analytics.
  • Query-based cost monitoring in AWS Cost Explorer.
🧠 How It Works?
  • Divide the array into segments (typically halves).
  • Store results of range queries in a binary tree format.
  • Update and query operations take O(log N) time.
