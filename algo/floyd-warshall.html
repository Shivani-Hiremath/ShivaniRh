Floyd-Warshall Algorithm

Category: All-Pairs Shortest Path

Description: Computes shortest paths between all pairs of vertices in a weighted graph.

📦 Business Case:

  • Optimizing delivery times between all warehouses.
  • Efficient for pre-computing shortest routes between multiple depots.
  • Can be used to analyze network latency or costs across many endpoints.
🧠 How It Works?
  • Initialize a matrix with direct distances between nodes (∞ if no edge).
  • Iterate using each node as an intermediate point and update shortest paths.
  • Uses Dynamic Programming concept: dp[i][j] = min(dp[i][j], dp[i][k] + dp[k][j])
