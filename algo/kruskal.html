Kruskal’s Algorithm

Category: Graph, Minimum Spanning Tree

Description: Finds a minimum spanning tree by adding the lowest-weight edge that doesn’t form a cycle.

📦 Business Case:

  • Designing cost-effective network connections between AWS data centers.
  • Building efficient logistics and warehouse networks.
  • Reducing infrastructure costs in distributed systems.
🧠 How It Works?
  • Sorts all edges by weight.
  • Adds edges one by one, skipping those that would form a cycle (using Union-Find).
  • Stops when all vertices are connected (n-1 edges for n nodes).
