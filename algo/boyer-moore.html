Boyer-Moore Algorithm

Category: Alexa Intent Matching

Description: Matches patterns right-to-left using the Bad-Character and Good-Suffix heuristics, often skipping large chunks of text, making it perfect for long voice-to-text commands.

📦 Business Case (Amazon):

  • Matching user voice commands to Alexa skill templates quickly.
  • Filtering long chat strings for offensive phrases in real time.
  • Searching large codebases in CodeCatalyst for specific function names.
🧠 How It Works?
  • Pre-compute two heuristic tables: Bad-Character & Good-Suffix.
  • Align pattern with text; compare from right to left.
  • On mismatch, shift pattern by max heuristic value (can skip many chars).

🧮 Complexity Analysis
  • Best Time: Sub-linear  ≈ O(n/m)
  • Worst Time: O(n m) (pathological)
  • Space: O(m + σ) — pattern + alphabet size
